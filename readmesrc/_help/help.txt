*readme.js ヘルプ[#200804041528]
**readme.js とは[#200804041543]
-もともとは公開ソフトのreadme用として作成したものですが、プロジェクトの情報共有の簡易ツールとして使用したところ、思いのほか好評でした。よろしければご活用ください。

-「readme.htm」と同じフォルダにある「readmesrc」内のファイル内容が表示されます。ファイル内容は、整形ルールに従って整形された上で表示されます。

-整形ルールはPukiWikiをベースにしていますが、すべてのルールが使用できるわけではありません。また、いくつかの制限があります。

-プロジェクトでは「メモページ」と呼んでいました。・・・ネーミング募集中です。


** 新しいページを作成するには?[#n2c2eb75]
***初級編[#20:57_2009/05/26]
-「readme.htm」と同じフォルダにある「readmesrc」にテキストファイルを作成します。拡張子は「txt」とし、文字コードはUTF-8で保存してください。

-各ファイルの先頭は以下のようにしてください。

 *readme.js ヘルプ[#200804041528]

-*で開始し、ページタイトルを記述した後、[#の後に全体で一意となるような文字列を記述し、]で閉じてください。ここでは日付+時刻を使用しています。

--文字列には空白を含めないでください。クエリパラメータによる直リンクがうまく動作しなくなります。

-*の大見出しは各ファイル先頭に1つだけ記述できます。ファイルの途中では記述できません。

-ファイルを作成した後、readme.htmをエディタで開き、以下の部分を編集します。

 function init(){
   new Readme([
     "sample1",
     "sample2" //←終端の「,」は不要
   ]);
 }

-上記3～4行目がファイル名指定部分です。拡張子は不要です。ここに記述された順にファイル内容が変換表示されます。追加する場合は「,」繋ぎで記述してください。終端の「,」は不要です。(つけるとエラーになります。)

***上級編[#20:58_2009/05/26]
-初級編の方法では「readmesrc」フォルダ直下にしかテキストファイルを配置できませんが、ここで説明する方法を使用すると、「readmesrc」フォルダ以下にさらにディレクトリを作成してグループ分けすることができます。

-readme.htmのinit関数を以下のように書き換えます。
 function init(){
   new Readme({
     page:{
       sample1:{
         src:[
           "src11",
           "src12"
         ]
       },
       sample2:{
         src:[
           "src21",
           "src22"
         ]
       }
     }
   });
 }

-このとき、フォルダ構成は以下のようになります。
 '''root'''
  ┃ readme.thm
  ┃
  ┗ readmesrc
      ┣ sample1
      ┃    src11.txt
      ┃    src12.txt
      ┃
      ┗ sample2
            src21.txt
            src22.txt

-ページを表示すると、上述のsample1以下のファイル内容が変換表示されます。
-また、ページ右上にプルダウンが表示され、sample1とsample2、および一覧、検索、ヘルプが選択可能となっています。選択することで、ページ内容が切り替わります。
-ファイルを追加する場合は各srcの配列にファイル名を追加してください。
-ディレクトリを追加する場合はpageにプロパティにオブジェクトを追加してください。プロパティ名はディレクトリ名に合わせてください。ディレクトリ名が全角文字等、プログラムで使用できない文字を含む場合は"か'で囲ってください。
-Readmeコンストラクタの引数オブジェクトや、各ディレクトリに対応するページオブジェクトにて指定できるプロパティは、これら以外にも様々なものがあります。[[プロパティマニュアル>#20101208164429]]を参照してください。
-Operaブラウザではpage以下に記述したディレクトリ順どおりに表示されない場合があります。([[後述>#20101209123655]]参照)


**目次作成機能[#200804051613]
-ページ上部の「ここをクリックすると目次を作成します」と表示されている要素をクリックすると目次を作成します。
-目次はすべて各見出しへのアンカーになっています。
-本機能は現在IEブラウザのみで動作します。
